Saved stations

This dialog is opened tapping the Station button in the Shot list window.

It displays

  • selected station (the "current" one by default)
  • Clear: clear the station field
  • Save: add the station to the list of saved stations
  • OK: set the station as "current"
  • Delete: drop the saved station
  • comment saved with the station

It also displays the list of saved stations.
Tapping an entry in the list displays it in the above form.
