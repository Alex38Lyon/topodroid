[7] SHOT LIST

The shots are shown in a list. The list has no header to save space. The shot informations are
  • Index (optional: it is shown only if the related choice in the Display mode dialog is checked)
  • From station
  • To station
  • Distance
  • Azimuth
  • Inclination
  • Extend direction: either left (<), right (>), vertical (|), or ignore (-)
Additional characters denote the presence of a comment (N), a photo (P), and a flag: either duplicate shot (*) or a surface shot (-).

Blank shots, and repeated-legs can be hidden. Splay shots can also be hidden to unclutter the list of shots. However, even if splays are not shown, you can see those at a station by tapping on the station name in a leg shot. To hide them tap again on the station name (even a tap on the name in a splay will do).

Topodroid can set the station names automatically according to a station assignament policy.

Shot edit dialog

If you choose not to have the station assigned automatically, you must set them manually. Tapping on a shot data brings up the shot edit dialog. With it you can set the station(s) (if the shot is blank TopoDroid suggest them for you incrementing those of the previous leg), the direction of extend for the profile (either left, right, vertical; if none is checked the shot is ignored in the profile), and the shot flag (whether it is a regular shot, or duplicate or surface: if no flag is set the shot is "regular"). Finally you can add a brief comment to the shot. The values of length, azimuth and inclination of shots downloaded from the DistoX or of an imported survey cannot be modified. For manually entered shots they are editable. The Reverse button swaps the stations. The arrow buttons let you move forward and backward over the leg shots. The OK button save the changes and closes the dialog. The Save button saves the changes but does not closes the dialog.

This dialog has six check buttons:

  1. duplicate leg: this checkbox marks the shot as duplicate, so that it does not count towards the cave statistics
  2. surface denotes a surface shot. Surface shots do not count for the cave statistics either.
  3. repeated leg tells TopoDroid that the shot is a "repeated leg"; the shot stations are blanked out and it is marked as "repeated leg".
  4. merge with leg tells TopoDroid that the shot is the first of a leg that has been assigned to the following shot.
  5. splay group is used if only the From station has been entered (and the To station is empty): it changes the From station of all the splay shots of the group. Effectively this assigns the group of splays from one station to another.
  6. renumber is used to rename subsequent shots; if the editing shot is a leg (ie, the From and To stations are specified) and the box is checked, all subsequent shots are renamed according to the current station assignment policy, their type not being changed (ie, splays remain splay, legs remain leg, and blank shots remain blank).

If you use TopoDroid automatic station assignment, you are likely to need the shot dialog only to enter the comment, and to change the "extend" direction. These two actions can be done also from the drawing window, by selecting the shot in "edit" mode [A] and picking the Note button: this is described in the section on the Sketch window).

This about covers all the actions one can do with shots, but you do not need to master all of them. The important things are really few: they amount to properly assign stations to the shots, maybe flags, and add comments.
