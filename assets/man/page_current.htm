Active station

The active station is the station from which the survey will continue. If you have set the active station, TopoDroid will continue numbering stations from it. If the active station is not set, the last one is used.

Thus you can start a branch in the middle of the survey by selecting the station you want to attach as active station. To set the active station long tap on the station name in the shot list (a normal tap toggles the display of the relevant splay shots when the display of splays is globally disabled). The active station is highlighted green in the shot data window. A long tap on another station switches the active station. A long tap on the active station turns it off. Alternatively, you can replace the from station of the first shot of the branch, afterwards.

The active station can be set in three ways

  • with a long tap on the station name in the shot data window.
  • through the Station edit dialog in the Sketch window.
  • selecting the station from the Saved stations dialog
