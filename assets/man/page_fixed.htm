Fixed point

This dialog is opened tapping a fixed point entry in the list if the Location dialog.

The dialog shows

  • longitude and latitude coordinates
  • altitude over the WGS-84 ellipsoid
  • altitude above mean sea level
  • fixed point name
  • the magnetic declination at the point position (at the current date)
  • View the position with a Location app (provided you have one installed)
  • Convert the location to another reference system (using the app Proj4)
  • Delete the fixed point record
The orthometric altitude is computed from the ellipsoidic altitude using the geoid model EGM2008.

The magnetid declination is computed using the World Magnetic Model for 2015-2020. To save it with the survey informations tap the Save button after computing it.
