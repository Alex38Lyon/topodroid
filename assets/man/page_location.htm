Location dialog

The "Location" button (the "GPS" icon) of the "Survey info" window opens the "Location" dialog.

Through this dialog you enter (or get with the GPS) the coordinates of a fixed station, eg, the entrance.

  1. first specify the station name.
  2. next enter the coordinate manually ("Add" button), or get them with the Android GPS ("GPS" button), or import them from Mobile Topographer ("Import" button).

To use the GPS the Android Location must be enabled. While searching for satellites the "GPS" button becomes red (or green) displays the number of satellites in sight. You should wait till the button becomes green and the coordinates are displayed underneath. Then stop the GPS location tapping the button again, and tap the "Add" button.

Internally TopoDroid stores the coordinates in decimal degrees. However, in the coordinates dialog, you can write latitude and longitude either in the format "dd:mm:ss.ss" or as decimal degrees. If your keyboard does not show the semicolon ':' use the space instead. If your keyboard does not have the decimal point '.' use the slash '/' instead. TopoDroid custom keypad has two buttons for "degrees" and "minutes".
Tapping the "E" and "N" buttons changes the East/West and North/South hemisphere for the coordinates. Effectively it is like entering negative coordinates.
The altitude is stored in meters and it is always the WGS84 altitude.

Underneath the buttons the dialog lists the coordinates of all fixed points, as "station longitude latitude altitude". Tapping any of them opens the "Fixed Station" dialog where you can

  • compute the magnetic declination at the station,
  • convert the coordinates to another coordinate reference system (using Proj4),
  • see the orthometric altitude
  • change the station name,
  • delete the fixed station record.

Location settings

  • Location units: either dd.mm.ss or decimal degrees [A]
  • C.R.S. coordinate reference system [default is long-lat] (disabled)
