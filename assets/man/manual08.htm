[8] SURVEY INFO WINDOW

A long tap on the survey entry in the main window opens the "Survey Info" window. Alternately you get here from the appropriate menu of the shot window.

BUTTONS

  • Notes: survey annotations dialog
  • Stats survey statistics dialog
  • 3D: the 3D display
  • GPS [A]: location dialog
  • Photos [A] lists the pictures of the survey. Tapping one of them you get a dialog to edit the description or delete the photo
  • Sensors [A]: shows the list of sensor measurements of the survey. At the moment these are not editable.

MENUS

  • Export: export dialog
  • Delete delete dialog
  • Calibration instruments calibration dialog
  • Settings (same as the shot window settings)
  • Help

Export dialog

The export dialog has a selection-list of export-formats: Compass, cSurvey, Survex, PocketTopo, Polygon, Therion, VisualTopo, and Walls. Furthermore the data can be exported as DXF, and CSV text file. If the survey has a fixed station it can be exported as KML, and track file (PLT). This export fails if no the survey station is fixed. Finally you can export a ZIP archive with all the data of the survey.

There are a number of export settings, that control the way TopoDroid writes the survey data in the various export formats.

Delete dialog

The delete dialog ask for confirmation to delete the survey. Deleting the survey is unrecoverable: the survey and all associated data (photos, notes, sketch files, etc. ) except the zip archive. Make sure to export a backup ZIP archive before you delete the survey.
