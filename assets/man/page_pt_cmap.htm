PocketTopo COLOR MAP

This dialog is opened selecting the PocketTopo color-map setting (in the Import/Export section).

It displays a table with three columns.

  • PocketTopo color
  • associated line (TopoDroid name)
  • associated point symbol (TopoDroid name)

The color-map is used in the import of ".top" files, to map PocketTopo colors to TopoDroid drawing items.
