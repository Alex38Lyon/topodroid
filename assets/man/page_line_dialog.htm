Drawing line dialog

You open the "Drawing line" dialog when you select a point of the a line (in editing mode of the "Sketch" window), and tap the "Edit" button (the "note and pencil").

This dialog lets you edit the properties of the line:

  • outline (either "in" or "out")
  • whether the line is normal or "reversed"
  • generic Therion options

With this dialog you can also

  • make the line more straight
  • reduce the number of points in the line
  • make the line closed
