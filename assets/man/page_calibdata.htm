Calibration data

This dialog is opened tapping on a data item in the Calibration data window.

Through this dialog the item group can be set or changed: use a positive integer for data that take part to the computation of the calibration coefficients, 0 (zero) for data that do not contribute to the computation, and -1 (negative one) for "deleted" data. Deleted data are not usually shown in the calibration data list and are not taken into account in the automatic group assignment.

The dialog has three buttons:

  • Regroup: if checked, when you save your changes, TopoDroid recomputes the items groups starting at that item, using the current group policy.
  • Save the group number and closes the dialog.
  • Delete the item


Deleted item are not dropped from the database. By default, they are not listed in the display. You can toggle their display with the Display menu. Deleted items are shown with a grey background. If you tap on a deleted item you get a dialog to undelete the item.
