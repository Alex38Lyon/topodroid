Sketch point dialog

The "Sketch point" dialog is opened when you select a point symbol and tap the "Item edit" button (the "note and pencil") in the "Sketch" window (in "edit" mode).

With this dialog you can change

  • the point size (one of XS, S, M, L, XL)
  • the point orientation, if the point is orientable
  • the point text, if the point has text associated
  • the point additional Therion options

The dialog has a Save button to confirm the changes and close it.

Use the "Back" key to close the dialog without confirming the changes.
