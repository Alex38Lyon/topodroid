Shot edit dialog (2)

The secondary "Shot edit" dialog is opened by long-tapping the shot entry in the "Shot list" window.

This dialog shows the shot stations and data (length, azimuth, inclination).

At an activity level "Basic" or "Normal" this dialog has only a button to Delete the shot.

At higher activity levels it has five buttons:

  • Photo: "New Photo" dialog.
  • Sensor: "New Measurement" dialog.
  • Add a shot after this (manually entering the data).
  • Split the survey at this shot.
  • Delete the shot.

The delete button does not remove the shot from the database. The shot gets marked as "deleted" and can be later recovered with the "Recovery" dialog.
