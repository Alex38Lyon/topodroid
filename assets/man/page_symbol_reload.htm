The drawing tools are distinguighed by the name and the type (point, line, or area).
A few tools are invluded in more than one set.

By default only the speleology tools are installed. The installation of additional tools is disabled by a setting [G].

With the PALETTE menu [G] of the main window you get a dialog listing the sets of tools. Check the tool sets you want to add and press the button Install.
If you press the button Replace the currently installed tools are removed before installing the tool sets you selected.

After installing you must enable the tools you need, using the Palette button of the main window. The choice of enabled tools is stored in the database and is not lost when tools are removed, therefore you will find them enabled if you re-install the set later.
