Mobile Topographer dialog

The "import" button of the "Location" dialog leads to the "Mobile Topographer" dialog.

With this dialog you can import in TopoDroid the coordinates of a point from a Mobile Topographer point list file.

  1. Open Mobile Topographer and find the point coordinates ("flag" icon on the left).
  2. When you have reached a sufficient accuracy in the coordinates, save the point (third icon "doc with plus").
  3. Go to the list of saved points (fourth icon "doc")
  4. Move the point you just got to the top with the green arrows, as TopoDroid imports the first point in the list.
  5. Save the list (fourth icon "diskette"). The file is saved in "MobileTopographer/pointlist"
  6. Import the coordinates in TopoDroid.
