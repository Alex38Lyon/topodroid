Calibration data dialog

The "Calibration data" dialog is opened tapping the data entry in the "Calibration data" window.

Theis dialog displays

  • azimuth, inclination, roll
  • calibration error
  • regroup: reassign the groups from this data onward
  • delete the data
  • Save the changes
Delete does not remove the data from the database, but it marks it with a negative group number (-1), so that it does not count for the calibration nor it is (normally) shown in the data list.
