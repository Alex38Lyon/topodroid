[6] SURVEY DATA WINDOW

The survey shot window displays the list of the shots of the survey.

TopoDroid differentiates among four kinds of shots:

  1. "blank" shots, ie, shots with the stations not set yet
  2. "splay": it has only one station, usually the "From" station
  3. "leg" (the first shot of a group of repeated measurements): this shot has both the "From" and the "To" station set
  4. "repeated-leg" (the following shots of a group of leg shots): these shots do not need to have the stations set

Each kind of shots is displayed with a different color: "legs" are white, "repeated-legs" grey, "splays" blue, "blank" red. Furthermore the background color is blueish for the last downloaded shots, and reddish for unreliable shots, ie. shots with "abnormal" field intensities (only DistoX2).

ACTIONS

  • tap on a shot: shot edit dialog
  • long tap on a shot: shot secondary actions dialog
  • tap on a station: show/hide splays at the station
  • long tap on a station: select/deselect the "current" station

BUTTONS

  • Download: DistoX data download. Hidden if no DistoX selected
  • Bluetooth: bluetooth dialog
  • Display: to choose the display mode
  • Sketch: sketch dialog. A long tap opens immediately the most recent sketch
  • Notes: survey annotations
  • Extend: extend reference
  • Stations [N]: saved stations
  • Add [N] manual shot data dialog

Sketch dialog

Through the Sketch dialog you create a new sketch or open an existing one. There is a button to create a new sketch, and a two-column list of the current survey sketches (items in the left column open a sketch with its plan view, the right column items open sketches in profile view; once open in a view you can switch to the other from within the sketch window).

If the 3D modelling is enabled there is also a button to create a new 3D model, and the existing 3D models are listed after the sketches.

MENUS

  • Survey info: survey information window
  • Recover: deleted item (shot or sketch) recovery
  • Photo [A]: pictures taken for this survey
  • Sensor [A]: sensor data taken for this survey
  • 3D: 3D survey display (need Cave3D)
  • DistoX [A]: device window
  • Settings
  • Help

SURVEY SETTINGS

  • Team: the default surveying team. If the value is not empty, it is used to set the "Team" field when you create a new survey. Write your name and that of the cavers you usually survey with.
  • Stations policy: select the automatic station assignment policy.
  • Station names specifies the type of the station names, either alphanumeric (default) or numeric [B].
  • Initial station name: the default first station name for new surveys. Can be overridden in the New survey dialog.
  • Thumbnail size: size of the photo thumbnails
  • DATA UNITS >>
    • Length units: either meters or feet
    • Angle units: either degrees or grads
  • SHOT DATA >>
    • Leg tolerance: the angular tolerance (percent) between consecutive shots of a leg
    • Minimum leg shots: minimum number of close consecutive shots for a leg.
    • Splay vertical threshold: the "extend" direction assigned to a splay shot depends on the angle it makes with the leg(s). When the angle is closer than this value to 90 degrees the splay gets a "vertical extend". Otherwise it gets the same extend as the leg, or the opposite, depending whether the angle it makes with the leg is smaller or larger than 90 degrees.
    • Leg vertical threshold. this is like Therion "vthreshold": the LRUD for manually entered shots are considered West-East-North-South when the shot inclination exceeds the vertical threshold
    • Shot extend: either fixed or relative to the extend reference
    • Loop closure error compensation. It is advisable not to do the compensation, as this allows to see graphically the misclosures, and spot possible survey errors
    • Backsight fields in the shot manual input dialog [N]
    • Magnetic anomaly compensation. It requires backsight auto-station policy
    • Timer: seconds of delay before measuring directions with the Android sensors
    • Beep volume: timer beep volume [N]
  • LOCATION settings
  • ACCURACY settings

To exit the survey data window double tap on the BACK button of your device.
