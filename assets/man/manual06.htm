[6] SURVEY DATA WINDOW

The survey shot window displays the list of the shots of the survey.

TopoDroid differentiates among four kinds of shots:

  1. "blank" shots, ie, shots with the stations not set yet
  2. "splay": it has only one station, usually the "From" station
  3. "leg" (the first shot of a group of repeated measurements): this shot has both the "From" and the "To" station set
  4. "repeated-leg" (the following shots of a group of leg shots): these shots do not need to have the stations set

Each kind of shots is displayed with a different color: "legs" are white, "repeated-legs" grey, "splays" blue, "blank" red. Furthermore the background color is blueish for the last downloaded shots, and reddish for unreliable shots, ie. shots with "abnormal" field intensities (only DistoX2).

ACTIONS

  • tap on a shot: shot edit dialog
  • long tap on a shot: shot secondary edit actions
  • tap on a station: show/hide splays at the station
  • long tap on a station: select/deselect the "current" station

BUTTONS

  • DistoX data download. Hidden if no DistoX selected
  • Bluetooth
  • Data display mode
  • Sketch list. A long tap opens immediately the most recent sketch
  • Survey notes
  • extend reference
  • saved stations
  • Add manual shot data dialog

Sketch dialog

Through the Sketch dialog you create a new sketch or open an existing one. There is a button to create a new sketch, and a two-column list of the current survey sketches (items in the left column open a sketch with its plan view, the right column items open sketches in profile view; once open in a view you can switch to the other from within the sketch window).

If the 3D modelling is enabled there is also a button to create a new 3D model, and the existing 3D models are listed after the sketches.

MENUS

  • Survey info: survey information window
  • Recover: deleted item (shot or sketch) recovery
  • Photo: pictures taken for this survey
  • Sensor: sensor data taken for this survey
  • 3D: 3D survey display (need Cave3D)
  • DistoX: device window
  • Settings
  • Help

SURVEY SETTINGS

  • Team: the default surveying team. If the value is not empty, it is used to set the "Team" field when you create a new survey. Write your name and that of the cavers you usually survey with. [default unset]
  • Stations policy: select the automatic station naming policy. [default "splays+foresight"]
  • Station names specifies the type of the station names, either alphanumeric or numeric [default "alphanumeric"]
  • Initial station name: the default first station name for new surveys. Can be overridden in the New survey dialog. [default unset]
  • Thumbnail size: size of the photo thumbnails [default 200, min 80, max 400]
  • DATA UNITS >>
    • Length units: either meters or feet [default meters]
    • Angle units: either degrees or grads [default degrees]
    • Sketch grid-cell unit: either meters ot yards [default meters]
  • SHOT DATA >>
    • Leg tolerance: the angular tolerance (percent) between consecutive shots of a leg [default 0.05, min 0]
    • Minimum leg shots: minimum number of close consecutive shots for a leg.[default 2, min 2, max 4]
    • Splay vertical threshold defines how the splay "extend" is set [default 10, min 0, max 90]
    • Leg vertical threshold (Therion "vthreshold"): the LRUD for manually entered shots are considered West-East-North-South when the shot inclination exceeds the vertical threshold [default 80, min 0, max 90]
    • Shot extend: either fixed or relative to the extend reference [default ???]
    • Loop closure error compensation. It is advisable not to do the compensation, as this allows to see graphically the misclosures, and spot possible survey errors [default "no"]
    • Backsight fields in the shot manual input dialog [N] [default "no"]
    • Magnetic anomaly compensation. It requires backsight auto-station policy [default "off"]
    • Timer: seconds of delay before measuring directions with the Android sensors [default 10, min 0]
    • Beep volume: timer beep volume [N] [default 50, min 10, max 100]
  • LOCATION settings
  • ACCURACY settings

To exit the survey data window double tap on the BACK button of your device.
