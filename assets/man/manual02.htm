[2] MAIN WINDOW

The main window has three (or five) buttons at the top, and the list of the surveys underneath. This list is empty until you create a survey. The first time you start the app a welcome message is shown. You can go back to it any time later with the About menu.

To exit TopoDroid main window double tap on the BACK button of your device.

ACTIONS

  • tap on a survey: survey data window
  • long tap on a survey: survey info window

BUTTONS

  • Device activity (DistoX icon): switches to the Device window
  • Add (a plus sign): new survey dialog
  • Import (a vertical arrow): survey or TopoDroid ZIP archive import
  • The optional buttons [E] are for the ThManager (Therion manager) and the SQLite viewer; both require external apps

MENUS

  • Palette allows to choose the drawing tools that are loaded.
  • Cosurvey [E] opens the dialog to connect with the TopoDroid app on another Android to make a join-surveying.
  • Settings: all the program settings.
  • Logs settings [E]. Logging is mostly for debugging app error.
  • About: the program welcome message
  • Help shows a brief descriptions of buttons and menus. At the top-right corner there is a button that opens this user manual. Each TopoDroid window has a similar Help menu.

SETTINGS

  • Work directory sets the current work directory. By default this is "TopoDroid".
  • Language overrides the system default language with a built-in language.
  • Text size of items in the lists
  • Buttons size, either "small" or "large": small buttons for a 5" screen, large buttons for 7" tablets.
  • Keyboard enables or disables the TopoDroid keypads.
  • Activity level selects the activity level [default "normal"].
  • Cosurveying [E] activate/deactivate the cosurveying service (very experimental).

If bluetooth is not on, the app asks whether to turn it on. Without bluetooth it cannot communicate with the DistoX and the survey data can only be entered manually. In this case the Download button in the Shot window is hidden, and that in the Sketch window is replaced by the Add button.
