Extend Reference dialog

The "Extend reference" dialog is opened by tapping the "azimuth" button either in the "Shot list" window or in the "Sketch" window.

The "Extend reference" is used to assign the profile "extend" direction to the midline legs.

The "azimuth" button displays either a long arrow inside a circle, oriented to the current extend reference direction, or a short arrow directed left or right (for fixed "extend").
When the profile "extend" is compute using the "extend reference" it is set to right if it differs from the reference by less than 90 degrees. Otherwise it is set to left.

The "Extend reference" dialog has

  • a button displaying the current selection of "extend reference"
  • a slider to change the value of the "extend reference", between -180 and 180 degrees
  • a "compass" button to use the Android magnetic sensors to get the "extend reference" azimuth
  • the left button to fix the "extend" to the left (-1)
  • the right button to fix the "extend" to the right (+1)
  • the ok button to save the "extend reference" azimuth
