Extend Reference

This dialog is opened tapping the azimut button either in the Shot list window or in the Sketch window.

The extend reference is used to assign the profile "extend" direction to the midline legs.

The azimut button displays either a long arrow inside a circle, oriented to the current extend reference direction, or a short arrow directed left or right (for fixed "extend").
When the profile "extend" is compute using the "extend reference" it is set to right if it differs from the reference by less than 90 degrees. Otherwise it is set to left.

This dialog has

  • "extend reference" azimut
  • slider to change its value between -180 and 180 degrees
  • compass sets the value using the Android magnetic sensors
  • left forces "extend" to LEFT (-1)
  • right forces "extend" to RIGHT (+1)
  • ok: use the azimuth for the "extend"
